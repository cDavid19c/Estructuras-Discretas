# Examen de Estructuras Discretas II

Aplicación web interactiva de una sola página con los 25 algoritmos del Examen Final de Estructuras Discretas II — Pontificia Universidad Católica del Ecuador (PUCE).

Cada algoritmo incluye descripción, cómo se ejecuta, ejemplo precargado, controles interactivos y análisis de complejidad. Funciona completamente offline: no requiere conexión, instalación ni dependencias externas.

**Autores:** Shirley Pilataxi · David Ocampo



## Contenido
Combinatoria y probabilidad, estructuras y recursión, grafos (BFS, DFS, Prim, Kruskal, Euler) y árboles (recorridos, BST, AVL).
